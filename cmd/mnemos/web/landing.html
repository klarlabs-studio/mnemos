Self-hosted memory + evidence layer for AI agents. Every recall ships with citations, every contradiction surfaces explicitly in the query response, every byte stays on your infrastructure. Drops into Claude Desktop, Cursor, LangGraph, or any MCP-compatible runtime in one config block.

$ mnemos process --text "Final pick is MySQL. Earlier we said Postgres."
related   1 relationship: contradicts  (trust 0.71 vs 0.83)

Write over HTTP from any language. Read back with claims + citations. Contradictions land as edges in the graph, not silently overwritten.

Works with Claude Desktop, Cursor, and LangGraph

mnemos mcp exposes 14 tools over stdio: query_knowledge, process_text, record_decision, query_lessons, which_test_to_trust, and more. Drop the snippet below into any MCP-compatible runtime — no SDK, no proxy.

~/.config/claude/claude_desktop_config.json 
{
  "mcpServers": {
    "mnemos": {
      "command": "mnemos",
      "args":    ["mcp"],
      "env":     { "MNEMOS_DB_URL": "sqlite:///Users/me/mnemos.db" }
    }
  }
}

Same surface available over REST (/v1/*) and gRPC for non-MCP runtimes.
